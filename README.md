# Gestão Consciente

Gestão Consciente é um aplicativo móvel em React Native projetado para ajudar os usuários a gerenciar suas finanças de forma eficaz. Ele fornece uma interface simples e intuitiva para rastrear despesas, definir metas de economia e visualizar hábitos de consumo.

## Funcionalidades

- **Tela de Boas-Vindas:** Uma tela de boas-vindas amigável que coleta o nome e o salário do usuário no primeiro acesso.
- **Resumo Financeiro:** Um resumo claro e conciso do saldo restante do usuário.
- **Rastreamento de Despesas:** Adicione e categorize facilmente as despesas como "necessárias" ou "não necessárias".
- **Gráfico de Rosca:** Um gráfico de rosca para visualizar a proporção de despesas necessárias vs. não necessárias.
- **Cotação de Moedas:** Um carrossel que exibe as taxas de câmbio mais recentes para BTC, USD e EUR.

## Começando

Para obter uma cópia local e executá-la, siga estes passos simples.

### Pré-requisitos

- Node.js
- npm
- React Native CLI
- Android Studio ou Xcode

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/GuilhermesNabor/GestaoConsciente
   ```
2. Instale os pacotes NPM
   ```sh
   npm install
   ```
3. Execute o aplicativo
   ```sh
   npx react-native run-android
   ```
